# Woodstockmanager

# 📦 Backoffice — Gestion de stock de bois

Application web de gestion de stock développée en équipe dans le cadre d'une formation. Elle permet de gérer les commandes et les clients via une interface frontend connectée à une API REST.

---

## 🛠️ Stack technique

**Frontend**
- Vue.js
- Pinia (gestion d'état)

**Backend**
- Laravel (API REST)

**Infrastructure**
- Docker

---

## ✨ Fonctionnalités

- Consultation de la liste des commandes
- Consultation de la liste des clients
- Affichage des détails via des modales

---

## 👥 Projet réalisé en équipe

Projet de formation réalisé en groupe composé de 5 personnes. Contribution personnelle principalement sur le frontend (Vue.js + Pinia).
